PDF to EPUB

Convert any PDF into a reflowable EPUB eBook. Ideal for reading on smartphones, tablets, and eReaders — with optional cover, metadata, and a generated table of contents.

How it works

  1. Extracts text and images from the PDF.
  2. Attempts to detect headings to build a Table of Contents (TOC).
  3. Reflows content into XHTML chapters and packages into an EPUB file.

Key Features

  • Produces standard EPUB 2/3 files compatible with most readers
  • Auto-generates TOC from detected headings
  • Option to upload a cover image and edit metadata (title, author, language)
  • OCR support for scanned PDFs (optional)

Use-cases

Convert course materials, research papers, guides, or self-published PDFs into EPUB for better on-device reading and distribution.

Example output

Title: Example eBook\nAuthor: John Doe\n\n[Chapter 1]\nThis is the start of chapter 1...\n\n[Chapter 2]\nContinuation...

Upload PDF

Accepted: PDF files up to 50MB (server-dependent). For best results, provide a text-based PDF rather than scanned images.

Notes for implementation

  • Server-side: Tools like ebooklib (Python), pandoc, or Calibre's ebook-convert are effective for building EPUBs.
  • To preserve tables and complex layouts, convert each page to HTML and post-process into chapters.
  • Use Tesseract OCR for scanned PDFs. Consider exposing an "OCR confidence" report to the user.
  • Generate EPUB metadata (title, author, language, identifiers) and let users override values in the form.
  • Optionally produce a MOBI/AZW3 output using Calibre for Kindle compatibility.
